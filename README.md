## 소개

웹 관련 공부로 시작하게된 첫 프로젝트인 스타벅스 코리아 웹사이트 클론코딩입니다. HTML, CSS, JS 등 바닐라로 진행하였으며 최대한 기본기 쌓기와 라이브러리 활용을 목표로 하였습니다.
